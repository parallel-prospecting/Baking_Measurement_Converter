This program converts various measurements used in baking

startAmount must be a number

startUnit and endUnit must be one of the following:

- 'tsp' //teaspoons 
- 'tbsp' //tablespoons 
- 'cups' 
- 'pints' 
- 'quarts' 
- 'gallons' 
- 'fl oz' //fluid ounces 
- 'mL' //milliliters
